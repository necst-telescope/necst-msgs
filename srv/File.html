necst_msgs.srv.File Reference

necst_msgs.srv.File

Data type for transferring file contents.

Fields

  • data (string) -- Contents of the file.
  • path (string) -- Where the original file was located, possibly with IP address.
  • --
  • success (bool) -- Whether the file was successfully transferred or not.

Example

>>> from from necst_msgs.srv import File
>>> msg = File.Request(data=path.read_text(path), path=path)
>>> response = File.Response(success=True)
