necst_msgs.srv.AuthoritySrv Reference

necst_msgs.srv.AuthoritySrv

Data type for authority management.

Fields

  • requester (string) -- Identifier of node requesting for privilege.
  • remove (bool) -- If true, this request is for privilege removal, not for acquisition.
  • --
  • privilege (bool) -- If true, the requester node is considered to be privileged.

Example

>>> from necst_msgs.srv import AuthoritySrv
>>> request = AuthoritySrv.Request(requester="test_node-123", remove=True)
>>> response = AuthoritySrv.Response(privilege=False)
