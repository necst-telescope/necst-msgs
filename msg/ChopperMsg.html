necst_msgs.msg.ChopperMsg Reference

necst_msgs.msg.ChopperMsg

Data type for chopper control commands.

Fields

  • insert (bool) -- If True, the absorber will cover the feed horn.
  • position (int) -- Motoric position of the chopper, especially useful to control some slider-type motors
  • time (float64) -- Time the command was sent.

Notes

Examples

msg = ChopperMsg(insert=True, position=4750, time=time.time()) publisher.publish(msg)

Raw definitions

bool insert
int position
float64 time
