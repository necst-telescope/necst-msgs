necst_msgs.msg.CoordArrayMsg

Data type for grouped celestial coordinate information.

Fields

  • lon (float64[]) -- Longitude in floating point value. Hour angle won't be accepted.
  • lat (float64[]) -- Latitude.
  • unit (string) -- Angular unit in which the longitude and latitude are given.
  • frame (string) -- Coordinate frame in which the longitude and latitude are given.
  • time (float64[]) -- Time the instructed coordinate si valid at, considering sidereal motion.
  • name (string) -- Name of celestial body.

Raw definitions

float64[] lon
float64[] lat
string unit
string frame
float64[] time
string name
