necst_msgs.msg.AntennaSectionStatus Reference

necst_msgs.msg.AntennaSectionStatus

High-level antenna control-section status for progress/diagnostics.

This message is additive. Existing ControlStatus remains the source used by legacy metadata gating. This status describes the section that corresponds to the current wall-clock/query time, not the farthest future command already published to the antenna drive queue.

Fields

  • active (bool) -- True when a current control section is known.
  • control_id (string<=64) -- Execution/control ID associated with this section.
  • section_uid (string<=96) -- Stable section identifier within the execution when available.
  • section_plan_index (int32) -- Planned section index inside a scan block, or -1 if N/A.
  • section_sequence_index (int32) -- Runtime section activation sequence, or -1 if N/A.
  • section_kind (string<=64) -- initial_standby, accelerate, line, decelerate, turn, ...
  • section_label (string<=64) -- Human-readable label, often an ON line label.
  • line_index (int32) -- 0-based scan-line index, or -1 if N/A.
  • section_start_unix (float64) -- Section start time [Unix seconds], if known.
  • section_stop_unix (float64) -- Section stop time [Unix seconds], if known.
  • section_duration_sec (float64) -- Section duration [s], if known.
  • query_time_unix (float64) -- Wall-clock time used to select the current section.
  • publish_time_unix (float64) -- Wall-clock time this status was published.
  • command_time_unix (float64) -- Command-time sample used for the status; normally query_time_unix.
  • nominal_fraction (float64) -- Command-time progress in this section, 0..1 when valid.
  • fraction_valid (bool) -- True when nominal_fraction is meaningful.
  • tight (bool) -- Strict/tight control requested for this section.
  • science_line (bool) -- True for science ON line sections.
  • interrupt_ok (bool) -- True when interruption is safe for this section.
  • geometry_valid (bool) -- True when planned section geometry fields are meaningful.
  • section_frame (string<=32) -- Coordinate frame of the planned section geometry.
  • section_unit (string<=16) -- Unit of geometry longitude/latitude values; degrees in v39.
  • section_start_lon_deg (float64) -- Planned start longitude [deg], if geometry_valid.
  • section_start_lat_deg (float64) -- Planned start latitude [deg], if geometry_valid.
  • section_stop_lon_deg (float64) -- Planned stop longitude [deg], if geometry_valid.
  • section_stop_lat_deg (float64) -- Planned stop latitude [deg], if geometry_valid.
  • section_speed_deg_per_sec (float64) -- Planned section speed [deg/s], if known.
  • status_basis (string<=32) -- current-time, horizon, idle, fallback, stale, ...
