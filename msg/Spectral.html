necst_msgs.msg.Spectral Reference

necst_msgs.msg.Spectral

Data type for spectral data.

Fields

  • data (float32[]) -- The spectral data.
  • g (strin) -- <=8 position # Observing position, such as ON, OFF, HOT, etc., up to 8 characters.
  • g (strin) -- <=16 id # Specific ID for current data, up to 16 characters.
  • time (float64) -- Time the data is taken on.

Example

>>> from necst_msgs.msg import Spectral
>>> msg = Spectral(
...     data=[1., 2., 3.], position="SKY", id="skydip_el=30" , time=1664792192.3645709
... )
