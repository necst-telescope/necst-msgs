necst_msgs.msg.Spectral Reference

necst_msgs.msg.Spectral

Data type for spectral data.

Fields

  • data (float32[]) -- The spectral data.
  • position (string<=8) -- Observing position, such as ON, OFF, HOT, etc., up to 8 characters.
  • id (string<=16) -- Coarse block / integration ID, up to 16 characters.
  • line_index (int32) -- Fine line index inside a scan block, or -1 if N/A.
  • line_label (string<=64) -- Human-readable fine line label, empty if N/A.
  • time (float64) -- Time the data is taken on.
  • time_spectrometer (string<=32) -- Time the data is taken from spectrometer, up to 32 characters.
  • ch (int32[<=2]) -- Stream range specification in spectrometer channel.
  • rfreq (float64[<=2]) -- Stream range specification in radio frequency.
  • ifreq (float64[<=2]) -- Stream range specification in intermediate frequency.
  • vlsr (float64[<=2]) -- Stream range specification in local standard of rest velocity.
  • integ (float64) -- Integration time.
