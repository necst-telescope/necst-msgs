necst_msgs.msg.ControlStatus Reference

necst_msgs.msg.ControlStatus

Interface definition for control status indicator.

Fields

  • controlled (bool) -- If true, this component has valid control command now.
  • tight (bool) -- Strict control is enabled or not.
  • remote (bool) -- If true, this component is remotely controlled
  • id (string<=64) -- Control section identifier.
  • interrupt_ok (bool) -- If true, this section isn't followed by bound tight control.
  • time (float64) -- Timestamp of this status
