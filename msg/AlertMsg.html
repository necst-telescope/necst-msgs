necst_msgs.msg.AlertMsg

Data type for alert information.

Fields

  • threshold (float64) -- Threshold of critical condition.
  • actual (float64) -- Actual value to compare with threshold.
  • warning (bool) -- If true, the situation nears the threshold.
  • critical (bool) -- If true, the situation has exceeded the threshold, and safeguard will be activated.
  • issuer (string) -- Publisher's node name, which indicate the type of alert.

Raw definition

# Data type for alert information.

float64 threshold  # Threshold of critical condition.
float64 actual     # Actual value to compare with ``threshold``.
bool warning       # If ``true``, the situation nears the threshold.
bool critical      # If ``true``, the situation has exceeded the threshold, and safeguard will be activated.
string issuer      # Publisher's node name, which indicate the type of alert.
