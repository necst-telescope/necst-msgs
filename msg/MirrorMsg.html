necst_msgs.msg.MirrorMsg Reference

necst_msgs.msg.MirrorMsg

Data type for mirror control commands.

Fields

  • position (string) -- Position of M4.
  • distance (float64) -- Move distance of M2.
  • status (int32[]) -- Position of M2.
  • time (float64) -- Time the command was sent.

Notes

Examples

msg = MirrorMsg(position="IN", time=time.time()) publisher.publish(msg)

Raw definitions

string position
float64 distance
int32[] status
float64 time
