necst_msgs.msg.SpectrometerStatus Reference

necst_msgs.msg.SpectrometerStatus

Lightweight spectrometer acquisition/recording status for progress/diagnostics.

This message intentionally carries no spectral array payload. It reports only acquisition, queue, metadata, and recording-health information.
