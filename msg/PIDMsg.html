necst_msgs.msg.PIDMsg

Data type for PID controller parameters.

Fields

  • k_p (float64) -- Coefficient of the proportional term.
  • k_i (float64) -- Coefficient of the integral term.
  • k_d (float64) -- Coefficient of the derivative term.
  • axis (string) -- Identifier of controlling axis, which will be az or el.

Example

>>> from necst_msgs.msg import PIDMsg
>>> msg = PIDMsg(k_p=2.0, k_i=0.0, k_d=0.0)

Raw definitions

float64 k_p
float64 k_i
float64 k_d
string axis
