necst_msgs.msg.TimedAzElInt64

Generic data type for set of int64 values for az/el axes, with timestamp.

Fields

  • az (int64) -- Value for azimuth axis.
  • el (int64) -- Value for elevation (altitude) axis.
  • time (float64) -- Time at which the data is published or is valid.

Raw definition

int64 el    # Value for elevation (altitude) axis.
